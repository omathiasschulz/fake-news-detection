Gensim

O Gensim começou como uma coleção de vários scripts Python em 2008, servindo
como base para gerar uma pequena lista dos artigos mais semelhantes a um determinado
artigo. Atualmente, o Gensim é uma biblioteca robusta do Python para a realização de
modelagem semântica não supervisionada a partir de texto sem formatação (Řehůřek,
2019).
